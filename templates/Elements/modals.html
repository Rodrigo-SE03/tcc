Adicionar Cargas

Método para adicionar cargas individualmente. Caso a carga desejada não possua fator ou não se deseje fazer a análise de reativos (grupo B) deve ser inserido o valor de 1. Os horários inseridos possuem precisão de 1 minuto, devendo estar de 00:00 a 24:00. Inserir um valor de início superior ao valor de final causará problemas no processamento dos dados.

Parâmetros da Análise

Esses elementos serão usados na análise do consumo de ponta, fora ponta e intermediário e dos resultados mensais. O horário de ponta será dado pelo intervalo entre o valor inserido e o fim das próximas 3 horas. Para inserir um horário de início não inteiro, como por exemplo 17:30, é necessário inserir 17.5.

Carregar Cargas

Essa função serve para carregar uma planilha Excel já construída contendo as cargas para a análise. Para obter a planilha base para o preenchimento das cargas basta clicar no botão "Baixar Modelos", presente na barra de navegação superior.

Salvar Análise

Caso todos os dados tenham sido preenchidos corretamente, ao pressionar o botão "Salvar", será gerado um arquivo Excel com o nome inserido na barra de preenchimento, contendo os resultados da análise. Caso não tenham sido preenchidos os dados de tarifa, será necessário ir à página de "Tarifas Praticadas" através da barra de navegação e preencher os valores.

Definir Tarifas

Nessa página serão preenchidos os dados de tarifa que serão aplicados na análise. Primeiramente é necessário selecionar o grupo da unidade consumidora (A ou B). Essa informação pode ser consultada na fatura de energia. A seguir deverão ser informados os valores das tarifas que serão usadas nos cálculos. Os valores das tarifas podem ser encontradas na fatura de energia ou através de informações disponibilizadas pela concessionária de energia local.

Carregar Tarifas

Essa função serve para carregar uma planilha Excel já construída contendo os valores das tarifas para a análise. Para obter a planilha base para o preenchimento das tarifas basta clicar no botão "Baixar Modelos", presente na barra de navegação superior.

Analisar Fatura

Essa análise só pode ser realizada para unidades consumidoras do grupo A. Para liberar os campos de preenchimento da análise é necessário selecionar o grupo A e preencher os dados das tarifas na página "Tarifas Praticadas", acessível através da barra de navegação.

Definir Demanda Contratada Atual

Preencher com os valores de demanda contratada atual da unidade consumidora. Caso a unidade esteja na modalidade verde, é necessário preencher o primeiro campo com o valor da demanda contratada e deixar o segundo campo com o valor igual a 0. É possível encontrar os valores de demanda contratada na fatura de energia.

Tipo de Preenchimento

O preenchimento manual requer que o usuário insira os dados de consumo e demanda das últimas 12 faturas de energia da unidade consumidora. O preenchimento automático requer que seja carregada a fatura de energia mais recente da unidade consumidora. Até o momento o preenchimento automático só pode ser aplicado para faturas de energia da Equatorial.

Carregar Histórico

Essa função serve para carregar uma planilha Excel já construída contendo os dados das faturas dos últimos 12 meses. Para obter a planilha base para o preenchimento dos dados basta clicar no botão "Baixar Modelos", presente na barra de navegação superior.

Carregar Fatura

Deve ser carregado nesse campo a fatura de energia mais recente da unidade consumidora. A fatura deve estar em formato PDF e deve ter sido retirada da plataforma da distribuidora de energia local para que o programa seja capaz de processar. Faturas fotografadas e digitalizadas não funcionarão. Caso haja algum erro na leitura, é recomendado usar o método de preenchimento manual.
